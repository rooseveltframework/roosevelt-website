    The easiest web framework
    to learn and build apps with

    Roosevelt is a web framework for Node.js based on Express that aims to reduce the complexity of your code as much as possible.

    Named for Teddy Roosevelt, the "trust buster" who curtailed the abuses of monopolists, the Roosevelt framework curtails the abuses of the merchants of complexity by trust busting (so to speak) all the unnecessary layers of abstractions and other needless complexities common to other web frameworks. Plus Teddy Roosevelt has a curly brace for a mustache.
